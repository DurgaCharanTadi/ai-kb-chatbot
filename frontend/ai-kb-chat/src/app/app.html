Knowledge Base AI-Chat

Welcome to the Codentis AI Chat — a full-stack web application that showcases how Retrieval-Augmented Generation (RAG) works with Amazon Bedrock. The chat panel (bottom-right) can answer questions grounded in a simulated enterprise knowledge base.

This demo uses an Angular 20 frontend and a Spring Boot backend integrated with Bedrock Knowledge Bases and S3 Vector storage.

💬 Click the chat icon in the lower-right corner to start exploring.

About Codentis

Disclaimer: Codentis is a fictional company created solely for demonstration purposes. It does not represent a real organization. All names, data, and financial figures are synthetic and exist to provide realistic context for this AI assistant.

Codentis is a simulated global technology corporation headquartered in Redmond, Washington. It is portrayed as one of the “Big Five” tech companies, developing enterprise software, cloud infrastructure, and artificial-intelligence solutions. The company’s narrative was designed to support realistic queries and responses for this AI knowledge-base chatbot.
